# 🤖 AI Code Reviewer

An intelligent web application that reviews code using AI and provides suggestions to improve code quality, readability, and best practices.

---

## 🚀 Features

* 🔍 Analyze code for errors and improvements
* 💡 AI-powered suggestions for better coding practices
* ⚡ Supports multiple programming languages
* 🧠 Detects code smells and inefficiencies
* 📝 Provides readable and structured feedback

---

## 🛠 Tech Stack

* **Frontend:** React.js, HTML, CSS, JavaScript
* **Backend:** Node.js, Express.js
* **AI Integration:** OpenAI API / LLM
* **Database:** MongoDB (optional)

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Channa2002/your-repo-name.git
cd your-repo-name
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Setup environment variables

Create a `.env` file and add:

```
OPENAI_API_KEY=your_api_key
```

### 4️⃣ Run the application

```bash
npm start
```

---

## 🧠 How It Works

1. User inputs code in the editor
2. Code is sent to backend API
3. Backend sends request to AI model
4. AI analyzes and returns suggestions
5. Results are displayed in UI
